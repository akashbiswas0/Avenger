# RentMyHeader - Neobrutalism Landing Page

A modern Next.js landing page with neobrutalism design style, featuring a navbar, hero section, and about/how-it-works section.

## Features

- **Neobrutalism Design**: Bold borders, high contrast, geometric shapes, and offset shadows
- **Minimalistic Theme**: Clean, simple layout with focus on content
- **Responsive**: Works on all device sizes
- **Next.js 14**: Built with the latest Next.js App Router

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Design Principles

The design follows neobrutalism principles:
- Thick black borders (4px)
- Offset box shadows (8px)
- High contrast colors
- Bold typography
- Simple geometric shapes
- Playful, energetic feel
